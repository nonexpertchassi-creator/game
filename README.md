# 🏭 방치형 공장 타이쿤 (Idle Factory Tycoon)

나사 한 개부터 시작해 로켓까지 찍어내는 제조업 재벌이 되는 방치형(idle) 게임입니다.
설치 없이 **`index.html`을 브라우저로 열기만** 하면 바로 플레이됩니다.

## 🎮 플레이 방법

1. 라인 아이콘을 **클릭**하면 생산 사이클이 돌고 돈이 들어옵니다.
2. 번 돈으로 라인을 **구매**하면 사이클당 수익이 커집니다.
3. **매니저**를 고용하면 그 라인이 자동으로 무한 생산합니다 (진짜 방치 시작 👔).
4. 이전 라인을 1개라도 보유하면 다음 라인이 **해금**됩니다.

## ✨ 특징

- **8단계 생산 라인**: 🔩 나사 → ⚙️ 볼트 → 🛠️ 기어 → 🔌 회로기판 → 🦾 로봇팔 → 🚁 드론 → 🚗 전기차 → 🚀 로켓
- **마일스톤 보너스**: 보유 수량이 25·50·100…개를 넘을 때마다 생산량 ×2
- **매니저 자동화**: 고용하면 손 떼도 알아서 돈 벌기
- **오프라인 수익**: 게임을 꺼둔 사이에도 최대 8시간까지 수익 누적
- **재투자(프레스티지)**: 누적 수익을 **명성**으로 환산해 리셋, 모든 수익 영구 ×(1 + 명성×2%)
- **구매 모드**: ×1 / ×10 / ×100 / MAX 한 번에 구매
- **자동 저장**: `localStorage`에 15초마다 저장 (서버 불필요)

## 🗂️ 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 마크업 |
| `style.css`  | 다크 네온 테마 스타일 |
| `game.js`    | 게임 로직 (순수 바닐라 JS, 의존성 0) |

## 🚀 실행

```bash
# 그냥 파일을 열거나
open index.html

# 로컬 서버로 띄우려면
python3 -m http.server 8000   # http://localhost:8000
```
